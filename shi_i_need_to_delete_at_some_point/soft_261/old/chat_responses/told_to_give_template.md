# 🔑 **What Counts as a “Good” Comment (Rubric Hack)**

To get full credit, each comment should include:

### 1. Specific

Bad:

> “The slides were confusing”

Good:

> “The slides contained dense blocks of text with no visual hierarchy”

---

### 2. Significant (WHY it matters)

> “This made it difficult to identify the key message”

---

### 3. Actionable (HOW to fix it)

> “Reducing text and emphasizing key points using headings would improve clarity”

---

### ✔ Put together:

> The slides contained dense blocks of text with no visual hierarchy, which made it difficult to identify the key message. Reducing text and emphasizing key points using headings would improve clarity.
