7\. Syntax error тАФ malformed fun form 

Expression: 

(*var* (*x* 5\) (*fun* ((*f y*) (+ *x y*)) (*apply* (*f* (*x*))))) 

Grammar Check 

The fun form requires: 

FAssign *тЖТ* ((FName FormalParams) Expr) 

where: 

FormalParams *тЖТ* () *|* (FormalParamList) 

In the given expression, the FAssign portion is ((*f y*) (+ *x y*)). The subpart (*f y*) should match (FName FormalParams). Here FName \= *f* and the formal parameters would be the bare identifier *y*. 

However, the grammar requires FormalParams to be either: 

┬И () тАФ the empty parameter list, or 

┬И (FormalParamList) тАФ a parenthesized list of variables. 

The bare identifier *y* (without surrounding parentheses) matches neither production. The cor rect form would be (*f* (*y*)). 

Syntax error. 

The formal parameter list in the function header is not enclosed in parentheses. (*f y*) does not match the grammar production for (FName FormalParams). We do not attempt any semantic derivation.
